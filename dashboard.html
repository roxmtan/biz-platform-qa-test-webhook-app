Setup Guide
To add a new app: Just point its webhook URL in Meta Developer Portal to:

https://qa-test-webhook-app.onrender.com/webhook/YOUR_APP_NAME

Example: /webhook/appqa5, /webhook/calling, /webhook/partners

No code changes needed! The app appears automatically in this dashboard.
